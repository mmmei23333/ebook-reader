# Ebook Reader 重构计划

## 基于截图的功能清单（排除iCloud）

### 交互逻辑变更
- 主页左滑 → 侧边菜单（个人中心）
- 侧边菜单覆盖在书库上方（截图可见右侧书架边缘）

### 侧边菜单（个人中心）
- 头像 + 个性标语（可编辑）
- 功能入口列表（圆角白卡片+图标+文字+箭头）：
  - 专业版
  - 设置
  - 外观
  - 菜单
  - 统计
  - 数据备份
  - 网盘和连接
  - TXT分章规则
  - 隐私安全
  - 使用指南
  - 建议反馈
  - 好评鼓励
  - 我的其他作品
  - 关于

### 设置页面
- 语言选择
- 漫画包预设、阅读方向
- epub自定义排版、忽略字体大小
- 启动时自动继续上次阅读
- 震动反馈开关
- 自动隐藏Home Indicator
- 购买后隐藏专业版入口
- 数据库修复
- 缓存管理

### 外观页面
- 外观模式（跟随系统/浅色/深色）
- 导航模式（经典侧滑）
- 主题色网格（9色+自定义）
- 界面字体
- 书架封面/特效/书脊/动画/时长显示

### 统计页面
- 日/周/月/年/总 时间维度
- 阅读时长、在读本数、读完本数、阅读次数、笔记数
- 今日阅读进度
- 阅读目标设置
- 阅读记录时间线

### 数据备份
- 自动备份开关
- 备份历史列表（时间+大小）
- 手动备份按钮
- 导出功能

### 网盘和连接
- WiFi传书
- 我的下载
- WebDAV/SMB/OPDS服务器
- 百度网盘/阿里云盘

### TXT分章规则
- 章节最大字数
- 正则规则管理
- 添加/临时生效/使用

### 隐私安全
- 手势密码
- Face ID
- 应用锁
- 隐藏书籍

### 菜单管理
- 书库/漫画 分类

### 升级专业版
- 月/年/买断方案
- 功能列表展示

## 实施顺序
1. 核心模型 + 服务层
2. 侧边菜单 + 主页左滑交互
3. 设置页面
4. 外观页面
5. 统计页面
6. 其他子页面
7. 升级专业版页面
