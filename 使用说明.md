# Ebook Reader - 使用说明

## 项目概述

这是一个简单的电子书阅读器Android应用，支持EPUB、PDF和TXT格式。

## 功能特性

- ✅ 导入本地电子书文件
- ✅ 书库管理
- ✅ 阅读进度保存
- ✅ 书签功能
- ✅ 笔记功能
- ✅ 夜间模式
- ✅ 字体大小调节
- ✅ 搜索功能

## 如何编译APK

### 方法一：使用GitHub Actions（推荐，无需本地安装）

1. 创建GitHub账号（如果没有）
2. 创建新的GitHub仓库
3. 将整个 `ebook_reader` 文件夹的内容上传到仓库
4. 进入仓库的 "Actions" 标签页
5. 工作流会自动运行并编译APK
6. 编译完成后，在 "Artifacts" 部分下载APK

### 方法二：本地编译（需要安装Flutter）

1. **安装Flutter SDK**
   - 访问 https://flutter.dev/docs/get-started/install
   - 下载并安装Flutter SDK
   - 配置环境变量

2. **安装Android Studio**
   - 下载并安装Android Studio
   - 安装Android SDK

3. **编译APK**
   - 双击运行 `build.bat`
   - 或者打开命令行，运行：
     ```
     flutter pub get
     flutter build apk --release
     ```

4. **获取APK**
   - 编译完成后，APK位于：
     `build/app/outputs/flutter-apk/app-release.apk`

## 如何使用

1. 将APK传输到安卓手机
2. 安装APK（可能需要开启"未知来源"安装权限）
3. 打开应用
4. 点击右下角的 "+" 按钮导入电子书
5. 点击书籍封面开始阅读
6. 阅读时点击屏幕中央显示控制栏

## 常见问题

**Q: 为什么无法安装APK？**
A: 请确保在手机设置中开启了"允许安装未知来源应用"。

**Q: 如何导入电子书？**
A: 点击右下角的 "+" 按钮，选择手机中的电子书文件。

**Q: 阅读进度会保存吗？**
A: 是的，阅读进度会自动保存。
